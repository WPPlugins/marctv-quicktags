== Description ==

Displays a bold, italic, add link and quote button on top of the comment form. . 

== Installation ==

Upload the "marctv_quicktags" folder into the plugins folder of your blog, Activate it. You're done!

== Frequently Asked Questions ==

= Why did you do this? =

Because needed a simple plugin to do this job and decided to do it on my own.
